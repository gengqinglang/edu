测试说明

此页面用于测试新增的当前阶段剩余年数费用计算功能，请验证以下效果：

  • 当前阶段费用卡片: 每个路径的第一张卡片显示当前阶段剩余费用
  • 剩余年数计算: 根据当前年级计算剩余年数（如高中1年级→剩余2年）
  • 费用计算准确: 剩余年数费用 = 剩余年数 × 年均费用
  • 路径总费用: 包含当前阶段剩余费用 + 后续阶段费用
  • 步骤编号调整: 当前阶段费用卡片为第0步，后续步骤编号相应调整
  • 特殊样式: 当前阶段费用卡片有特殊的绿色边框和标签
测试用例说明
  • 高中1年级 → 大学: 应显示高中剩余2年费用 + 大学4年费用
  • 高中2年级 → 大学: 应显示高中剩余1年费用 + 大学4年费用
  • 高中3年级 → 大学: 应显示0年剩余费用 + 大学4年费用
  • 小学4年级 → 研究生: 应显示小学剩余3年费用 + 初中3年 + 高中3年 + 大学4年 + 研究生3年
